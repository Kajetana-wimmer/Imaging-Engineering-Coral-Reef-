# Imaging-Engineering-Coral-Reef-

The system images coral reefs using underwater pictures taken in marine environments.  It must classify coral reefs into three categories based on visual characteristics, especially the colors; colorful (healthy coral), white (bleached coral), brown-grey-black (dead coral)
The imaging system will answer: what is the life stage of the coral reefs present in the image: healthy, blanched or dead?
The primary output of the system will be a decision (a classification) of the coral reef life stage for each image: healthy, bleached or dead. 
If we have time, the system could produce a spatial heat map of coral reef life stages across a reef area using aerial images. This would enable a larger visualisation of reef life stages.
